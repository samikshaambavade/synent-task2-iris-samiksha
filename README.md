# Iris Data Visualization

## Objective
To visualize patterns in the Iris dataset using Python.

## Dataset
Iris Dataset (Kaggle)

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Visualizations
- Bar Chart
- Histogram
- Scatter Plot
- Pair Plot

## Key Insights
- Setosa is clearly separable from the other species.
- Petal length and petal width are strongly correlated.
- Virginica generally has the largest petal dimensions.
